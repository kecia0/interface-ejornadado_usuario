# 🐶 Projeto AuPet

Projeto de design no Figma para uma aplicação de adoção de cães, focada em conectar abrigos a adotantes de forma simples e intuitiva. Inclui protótipos, fluxos e layouts pensados para incentivar a adoção responsável.
